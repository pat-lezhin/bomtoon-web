## live 활성화 하기
npm run dev

## github action 과 동일한 프리뷰 보기
npm run build
npm run preview


## github 에다가 push 하기
git add .
git commit -m "Add background images and icon setup"
git push origin main

git config --global user.name "pat-lezhin"
